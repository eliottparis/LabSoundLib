# LabSoundLib

[LabSound](https://github.com/LabSound/LabSound) compiled framework and dependent libraries for iOS.

### Build

```console
./scripts/build_ios.sh
```

### Validate Pod

```console
pod lib lint --allow-warnings --no-clean --verbose
```
